Calling Java Code

The following shows an example of calling Java Code using the axel:code action.

There are three code calls on the page which are:
Get Total Memory Available<axel:code call="org.xmlactions.common.system.JS.getTotalMemory"/>
Get Used Memory<axel:code call="org.xmlactions.common.system.JS.getUsedMemory"/>
Get Environment Variable <axel:code call="org.xmlactions.common.system.JS.getEnv">
    <axel:param value="COMPUTERNAME"/>
</axel:code>
